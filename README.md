# Gemba Walk — Мобильное приложение для обхода производства

## Описание

Интерактивное React-приложение для проведения Gemba Walk (обхода рабочих мест) на производстве. Инструмент для аналитиков и собственников предприятий.

## Функции

- **Автосохранение** — данные сохраняются в localStorage браузера
- **Десктоп-макет** — боковое меню, двух- и трёхколоночные сетки
- **Разделение ролей** — вопросы для Аналитика и Собственника
- **Экспорт в PDF** — красиво сверстанный отчёт
- **Разделы**: Главная, Владелец, Станции (ЦПУ, Сварка, Покраска, Дерево, Сборка, Склад), Рабочие, Треугольник, РПА, Действия

## Стек технологий

- React (JSX)
- Inline Styles
- localStorage для хранения данных

## Файлы

- `gemba-mobile.jsx` — основной компонент приложения

## Как использовать

1. Встройте компонент в React-приложение (Vite / Create React App)
2. Откройте в браузере
3. Заполняйте данные по каждому разделу
4. Экспортируйте отчёт в PDF
